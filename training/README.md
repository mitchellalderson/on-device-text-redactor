# PHI Firewall - Fine-Tuning Pipeline

Fine-tune LFM2.5-350M for PHI redaction using LoRA on Modal GPUs.

All commands run from the project root via `npm run`.

## Prerequisites

- [Node.js](https://nodejs.org/) (for `npm run` commands)
- [uv](https://docs.astral.sh/uv/getting-started/installation/) (for leap-finetune)
- [Modal](https://modal.com/) account with billing set up
- [HuggingFace](https://huggingface.co/) account with write access

## One-Time Setup

```bash
# Clone and install leap-finetune
cd training
git clone https://github.com/Liquid4All/leap-finetune
cd leap-finetune && uv sync && cd ..
cd ..

# Install Python dependencies (in leap-finetune venv)
training/leap-finetune/.venv/bin/pip install pyarrow datasets huggingface-hub peft transformers torch onnxscript onnxruntime

# Authenticate
training/leap-finetune/.venv/bin/python -c "from huggingface_hub import login; login()"
modal setup
```

## Pipeline Commands

All commands run from the project root:

### Step 1: Prepare the Dataset

```bash
npm run prepare:data
```

Downloads the [Nvidia Nemotron-PII](https://huggingface.co/datasets/nvidia/Nemotron-PII) dataset, converts to SFT JSONL, and uploads to HuggingFace.

Options:

```bash
npm run prepare:data -- --sample 50000 --no-push
```

### Step 2: Fine-Tune on Modal

```bash
npm run train
```

Runs 3 epochs of LoRA SFT on a single H100. Set `detach: true` in `training/config.yaml` to survive laptop sleep.

Monitor detached jobs:

```bash
npm run train:logs
```

Config is in `training/config.yaml`:
- **Model**: `LFM2.5-350M` (350M instruction-tuned)
- **Dataset**: `aldersondev/phi-redaction-sft` (from Step 1)
- **LoRA**: default rank, targeting attention + MLP projections
- **Training**: 3 epochs, batch size 8, learning rate 2e-4
- **GPU**: 1x H100 on Modal

### Download checkpoints

```bash
modal volume ls phi-firewall-finetune
modal volume get phi-firewall-finetune /outputs/phi-firewall-redaction/<run-name> training/output/lora
```

### Step 3: Export to ONNX

```bash
# Merge + export + push to Hub
npm run export:model -- --lora-path training/output/lora --output-path training/output/merged --export-onnx --push-to-hub --tag v2-50k

# Skip merge (already merged)
npm run export:model -- --lora-path training/output/lora --output-path training/output/merged --export-onnx --skip-merge --push-to-hub --tag v2-50k
```

### Step 4: Deploy

```bash
npm run deploy
```

Builds the Vite app and deploys to Modal.

## Troubleshooting

- **"WebGPU not available"**: Enable at `chrome://flags/#enable-unsafe-webgpu`
- **"LoRA checkpoint not found"**: Check `modal volume ls phi-firewall-finetune`
- **HF upload 401**: Run `training/leap-finetune/.venv/bin/python -c "from huggingface_hub import login; login()"`
- **ONNX export fails**: Ensure `onnxscript` is installed
- **Modal timeout**: Increase `timeout` in `config.yaml` (default is 2 hours)
- **Detached job canceled**: Use `modal run --detach` (the `detach: true` config handles this automatically)
